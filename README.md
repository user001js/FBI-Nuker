# Lista de comandos - FBI Nuker
- `.` = prefixo (exemplo/padrão)
- `<>` = obrigatório
- `[]` = opcional

# 1. Nuke (destruir servidor)
`.1`, `.nk`, `.nuke`, `.atacar`, `.dest`, `.destruir`, `.attack` **ou** `.n` **+** `[banir 1/0]` `[seu texto]`

# 2. Banir todos os membros do servidor
`.2`, `.be`, `.baneveryone`, `.banall`, `.banirtodos` **ou** `.ba`

# 3. Expulsar todos os membros do servidor
`.3`, `.ke`, `.kickeveryone`, `.expulsartodos`, `.kickall` **ou** `.ka`

# 4. Renomear todos os membros do servidor
`.4 chen`, `.ra`, `.renameall`, `.renall`, `.r.all`, `.renomeartodos`, `.rt` **+** `<novo nome de usuário>`

# 5. Enviar mensagem direta para todos os membros do servidor
`.5 .dme`, `.md.all`, `.dm.all`, `.dmall`, `.mdall` ou `.dm.everyone` **+** `<mensagem>`

# 6. Enviar mensagem(ns) em todos os canais (spam all)
`.6`, `.sa`, `.sac`, `.spam.allchannels`, `.spam.todososcanais`, `.spamchannels`, `.sc.all` **+** `<quantidade de mensagens` `<texto da mensagem a ser enviada em todos os canais>`

# 7. Enviar mensagem(ns) no canal atual (spam to current channel)
`.7`, `.sc`, `.spamchannel`, `.stc`, `.scc` ou `.spam.canalatual` **+** `<quantidade de mensagem(ns)>` `<texto da(s) mensagem(ns)>`

# 8. Deletar todos os canais do servidor
`.8`, `.dch`, `.delete.all.channels`, `.delallchannels`, `.dac`, `.deleteall.channels`

# 9. Deletar todos os cargos do servidor
`.9`, `.dr`, `.delallroles`, `.dar`, `.deleteroles`, `.delroles` **ou** `.delete.all.roles`

# 10. Criar canais no servidor (spam channels)
`.10`, `.sch`, `.spam.channels`, `.createchannels` **ou** `.criarcanais` **+** `<quantidade de canais>` `<nome dos canais>`

# 11. Criar cargos no servidor (spam roles)
`.11`, `.sr`, `.spamroles`, `.swr`, `.criarcargos`, `.createroles` **ou** `.cr` **+** `<quantidade de cargos>` `<nome dos cargos>`

# 12. Editar ícone do servidor
* Envie um anexo/link de uma imagem

`.12`, `.si`, `.editservericon`, `.editicon`, `.iconserver` **ou** `.edi` **+** `<link/anexo/imagem>`
  
# 13. Editar nome do servidor
`.13`, `.sn`, `.edm`, `.editservername`, `.editarnome`, `.editnameserver`, `.servername` ou `.editname` + `<novo nome do servidor>`
  
# 14. Obter cargo com a permissão de "Administrador"
`.14`, `.ga`, `.getadmin`, `.obteradmin`, `.conseguiradmin` + `<nome do cargo>`

# 15. Recuperar (somente em mensagem direta com o bot)
* Cria 1 canal de texto no servidor de preferência

`.15`, `.rg`, `.recuperarservidor`, `.rs` ou `.recuperateserver` + `<ID do servidor>`

# 16. Configurações
`.settings`, `.config`, `.configurações` ou `.cfg`

# Como usar o **FBI Nuker v3.0** OTY5ODM4OTg3NjIxNTcyNjc4.GdRRvD.brst8vAwP0nZOecAB8uxYoXxUhuBNHL6G3_5oY

### 1. Após aparecer o painel mostrado acima, você precisará colocar o Token do seu bot
* Não sabe como criar um bot? Clique [aqui](https://youtube.com/) para assistir um tutorial de como criar seu bot.

![](https://user-images.githubusercontent.com/101312928/170846760-13e69cbf-31d9-4357-a3a6-81420e5c7124.png)

### 2. Insira um **prefixo** para o seu bot
O prefixo padrão do FBI Nuker é `.`, mas você pode inserir qualquer outro prefixo. Exemplo: `!`, `-`, `FBI.`, `>`, `<`, `?` ou qualquer outro de sua preferência.

![](https://user-images.githubusercontent.com/101312928/170846840-fda7cba6-f6c2-49db-8a31-720a403790d7.png)

### 3. Insira o ID do(s) administrador(es)/proprietário(s)

![](https://user-images.githubusercontent.com/101312928/170846926-26c1f687-3ba3-45ff-94f8-3a41063bc72d.png)

#### Ao inserir um ID de algum usuário, ele terá permissão para alterar as configurações abaixo.

![](https://user-images.githubusercontent.com/101312928/170846998-ea679a37-912f-4a09-a93a-aa1bc3b4b66f.png)

### 4. Insira `y` ou `s` se deseja criar o arquivo `config.json` ou insira `n` se não quiser criar o arquivo `config.json`.

- Se você não quiser criar o arquivo `config.json`, na próxima vez que você executar o FBI Nuker, você terá que inserir as opções novamente.

- Se você criar o arquivo config.json, você não precisará inserir as opções (Token, prefixo, ID) novamente. O config.json irá fazer esse trabalho para você!

- Se caso você quiser trocar o Token ou o prefixo do bot ou adicionar novos proprietários/administradores, basta excluir o arquivo `config.json`, executar o FBI-Nuker e inserir as opções novamente.

### 5. Aguarde alguns instantes até que você receba os dados do seu bot, como mostrado abaixo

![](https://user-images.githubusercontent.com/101312928/170847631-1001b4a0-61be-4cce-a718-5819a0e3e9b0.png)

### 6. Após receber uma confirmação como mostrado acima, você deve deixar o programa aberto e poderá abrir o Discord
