# Lista de comandos - FBI Nuker
- `.` = prefixo (exemplo/padrão)
- `<>` = obrigatório
- `[]` = opcional

# 1. Nuke (destruir servidor)
`.1`, `.nk`, `.nuke`, `.atacar`, `.dest`, `.destruir`, `.attack` **ou** `.n` **+** `[banir 1/0]` `[seu texto]`

# 2. Banir todos os membros do servidor
`.2`, `.be`, `.baneveryone`, `.banall`, `.banirtodos` **ou** `.ba`

# 3. Expulsar todos os membros do servidor
`.3`, `.ke`, `.kickeveryone`, `.expulsartodos`, `.kickall` **ou** `.ka`

# 4. Renomear todos os membros do servidor
`.4 chen`, `.ra`, `.renameall`, `.renall`, `.r.all`, `.renomeartodos`, `.rt` **+** `<novo nome de usuário>`

# 5. Enviar mensagem direta para todos os membros do servidor
`.5 .dme`, `.md.all`, `.dm.all`, `.dmall`, `.mdall` ou `.dm.everyone` **+** `<mensagem>`

# 6. Enviar mensagem(ns) em todos os canais (spam all)
`.6`, `.sa`, `.sac`, `.spam.allchannels`, `.spam.todososcanais`, `.spamchannels`, `.sc.all` **+** `<quantidade de mensagens` `<texto da mensagem a ser enviada em todos os canais>`

# 7. Enviar mensagem(ns) no canal atual (spam to current channel)
`.7`, `.sc`, `.spamchannel`, `.stc`, `.scc` ou `.spam.canalatual` **+** `<quantidade de mensagem(ns)>` `<texto da(s) mensagem(ns)>`

# 8. Deletar todos os canais do servidor
`.8`, `.dch`, `.delete.all.channels`, `.delallchannels`, `.dac`, `.deleteall.channels`

# 9. Deletar todos os cargos do servidor
`.9`, `.dr`, `.delallroles`, `.dar`, `.deleteroles`, `.delroles` **ou** `.delete.all.roles`


# 10. Criar canais no servidor (spam channels)
`.10`, `.sch`, `.spam.channels`, `.createchannels` **ou** `.criarcanais` **+** `<quantidade de canais>` `<nome dos canais>`

# 11. Criar cargos no servidor (spam roles)
`.11`, `.sr`, `.spamroles`, `.swr`, `.criarcargos`, `.createroles` **ou** `.cr` **+** `<quantidade de cargos>` `<nome dos cargos>`

# 12. Editar ícone do servidor
* Envie um anexo/link de uma imagem

`.12`, `.si`, `.editservericon`, `.editicon`, `.iconserver` **ou** `.edi` **+** `<link/anexo/imagem>`
  
# 13. Editar nome do servidor
`.13`, `.sn`, `.edm`, `.editservername`, `.editarnome`, `.editnameserver`, `.servername` ou `.editname` + `<novo nome do servidor>`
  
# 14. Obter cargo com a permissão de "Administrador"
`.14`, `.ga`, `.getadmin`, `.obteradmin`, `.conseguiradmin` + `<nome do cargo>`

# 15. Recuperar (somente em mensagem direta com o bot)
* Cria 1 canal de texto no servidor

`.15`, `.rg`, `.recuperarservidor`, `.rs` ou `.recuperateserver` + `<ID do servidor>`

# 16. Configurações
`.settings`, `.config`, `.configurações` ou `.cfg`
